# MERN E-Commerce Platform (Ongoing)

A full-stack MERN e-commerce application with admin product management.

## Features
- JWT Authentication & RBAC
- Admin product CRUD
- Image uploads via Cloudinary
- Redux Toolkit for state management

## Tech Stack
React, Vite, Redux Toolkit, Node.js, Express.js, MongoDB, Tailwind CSS
